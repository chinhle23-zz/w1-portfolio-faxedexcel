Dog Adoption

The goal of this project was to build a one page website for an animal shelter. I learned about CSS and its properties and syntax. I leveraged that knowledge with my HTML skills to create a more attractive website. I created radio buttons, check boxes, rounded borders, and overlayed images with translucent colors and text. I learned about margin, border, padding, and content width and how I can use flexbox to allow the website to dynamically change to fit any screen size and still look aesthetically pleasing.
